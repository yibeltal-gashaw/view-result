🎓 Student Result Telegram Bot

A Telegram bot built with Node.js, Telegraf, and MongoDB that allows students to securely retrieve their exam results by sending their Student ID.

🚀 Features
---
📊 Fetch student results from MongoDB

🤖 Telegram bot interface

🖼️ Welcome message with image

🔍 Case-insensitive Student ID search

⚡ Fast and lightweight

☁️ Ready for deployment on Render

🛠️ Tech Stack
---
Node.js

Telegraf (Telegram Bot API)

MongoDB + Mongoose

dotenv

Render (Deployment)
